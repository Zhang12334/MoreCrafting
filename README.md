# MoreCrafting/更多合成数据包
## What is it
A Minecraft Datapack , added more crafting recipes<br>
## How to use it
put it(MoreCrafting-Vx.x.x.zip) in ur world/datapacks folder , then /reload or /datapack enable "Files/MoreCrafting-Vx.x.x.zip"<br>
